DxfFlowchartMiya

直感的な操作と強力なDXF合成機能を備えた、プロ仕様のデスクトップ向けフローチャート作成アプリです。
インストール作業は不要。ダウンロードした実行ファイル（.exe）をダブルクリックするだけで、すぐにご利用いただけます。

🌟 主な機能

DXF図枠の直接合成: お手持ちのDXF図面枠（タイトルブロック等）をキャンバスの背景として読み込み、その「枠内」を確認しながらフローチャートを作成できます。出力時には、見た目の通りに合成されたDXFファイルが生成されます。

図枠の移動とロック: 読み込んだ図枠は自由に移動して位置合わせができ、ロック機能で誤操作を防ぐことも可能です。

多彩な図形と直角配線: 「処理」「分岐」「データ」などの図形と、美しく繋がる直角配線（Orthogonal Routing）をサポート。

自動階層レイアウト: 散らばったノードを一瞬で綺麗なツリー構造に自動整列させます。

仕様書自動生成: フローチャートの繋がりを解析し、Excelファイル（.xlsx）として手順リストを出力。

豊かなエディタ機能: Undo/Redo、コピペ、グループ化、配置整列、グリッド吸着など、CADのような操作感を実現。

幅広い出力形式: DXF、SVG、PNG、JPEG、PDF（ベクター画質）、Mermaid記法、Draw.io互換、Jw_cadコピーに対応。

📥 実行方法

配布元から DxfFlowchartMiya-v1.0.0.exe をダウンロードします。

ダウンロードした .exe ファイルをダブルクリックして起動します。

⚠️ 初回起動時に「WindowsによってPCが保護されました」と表示された場合

これはインターネットからダウンロードした新しいアプリに対する標準的な警告です。以下の手順で安全に起動できます：

メッセージ内の 「詳細情報」 をクリックします。

右下に表示される 「実行」 ボタンをクリックするとアプリが起動します。

💡 基本的な使い方

DXF図枠をセットする: メニューの「ファイル」>「DXF出力の図枠リンクを設定...」から図面枠ファイルを選択します。背景に表示された図枠に合わせてフローチャートを配置してください。ツールバーの「鍵アイコン」で図枠のロックを解除すれば、図枠自体の位置調整も可能です。

図形を置く: ツールバーから図形の種類を選び、キャンバスをクリックして配置します。

線を繋ぐ: 「接続」ツールを選び、元となる図形と先の図形を順番にクリックします。

文字を入れる: 図形や線をダブルクリックすると、内容を編集できます。

DXFで書き出す: 「ファイル」>「DXFエクスポート...」を実行すると、図枠とフローチャートが合成された状態で保存されます。
