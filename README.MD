DxfFlowchartMiya

直感的な操作と豊富なエクスポート機能を備えた、プロ仕様のデスクトップ向けフローチャート作成アプリです。
インストール作業は不要。ダウンロードした実行ファイル（.exe）をダブルクリックするだけで、すぐにご利用いただけます。

🌟 主な機能

DXF図枠(CADベース)連携: お手持ちのDXFファイルを「図枠」として読み込み、その上にフローを描画できます。エクスポート時には自動で元の図面データと統合され、CAD上でもレイヤーが分かれた状態で美しく出力されます。

多彩な図形と直角配線: 「処理」「分岐」「データ」などの図形と、美しく曲がる直角配線（Orthogonal Routing）をサポート。

自動階層レイアウト: バラバラに置かれたノードを、一瞬で綺麗なツリー構造に自動整列させます。

仕様書自動生成: フローチャートの繋がりを解析し、手順書としてExcelファイル（.xlsx）を自動出力。

モダンUI対応: 美しいベクターアイコンと、OS設定に連動する目に優しいダークテーマ/ライトテーマに対応。

豊富なエクスポート形式: DXF、SVG、PNG、JPEG、PDF（ベクター画質劣化なし）、Mermaid記法（エンジニア向け）で出力可能。Jw_cadへのダイレクトコピーにも対応。

Draw.io互換: 作成した図面をDraw.io（XML形式）として相互にやり取りできます。

📥 ダウンロードと実行方法

配布元から DxfFlowchartMiya.exe をダウンロードします。

ダウンロードした .exe ファイルをダブルクリックして起動します。

⚠️ 初回起動時に青い警告画面（Windows Defender SmartScreen）が表示された場合

これは、インターネットからダウンロードした新しいアプリに対するWindowsの標準的な安全確認画面です。以下の手順で安全に起動できます：

メッセージ内の 「詳細情報」 をクリックします。

右下に現れる 「実行」 ボタンをクリックします。

💡 基本的な使い方

DXFを図枠にする: 左上メニューの「ファイル」>「DXFを図枠(ベース)としてインポート...」からお手持ちのDXFファイルを選択すると、図面がキャンバスの原点に固定配置されます。不要になった場合は「DXF図枠をクリア」から削除できます。

図形の配置: 上部のツールバーから「処理」「分岐」などの図形アイコンを選び、キャンバスの好きな場所をクリックして配置します。

線の接続: ツールバーの「接続（リンクアイコン）」を選び、1つ目の図形をクリックした後、2つ目の図形をクリックして繋ぎます。

文字の入力: 配置した図形や線をダブルクリックすると、自由にテキストを入力できます。

スタイルの変更: 図形や線を選択した状態で、ツールバーから「背景色」「線の色」「線幅・線種」「矢印の向き」を細かくカスタマイズできます。

保存と書き出し: メニューの「ファイル」から、独自の形式（.json）での保存や、DXF・PDF・画像としてのエクスポートが可能です。
